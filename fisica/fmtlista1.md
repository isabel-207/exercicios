# 📘 Lista 1 – Trabalho e Energia

## 🔹 Exercício 1

O trabalho realizado por uma força constante sobre uma partícula durante um deslocamento retilíneo é positivo ou negativo:

a) se o ângulo entre for de 30°  
b) se o ângulo for de 100°  
c) F = (2,00i - 3,00j) N e d = -4,00i m  

---

## ✏️ Resolução

Sabemos que o trabalho é dado por:

W = F · d = F d cosθ

### a) θ = 30°
cos(30°) > 0 → Trabalho positivo

### b) θ = 100°
cos(100°) < 0 → Trabalho negativo

### c)
F = (2i - 3j)  
d = (-4i)

Produto escalar:
W = (2)(-4) + (-3)(0)  
W = -8 J  

→ Trabalho negativo

---

## ✅ Resposta:
a) positivo  
b) negativo  
c) negativo
